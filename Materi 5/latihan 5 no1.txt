Tulislah algoritma yang lengkap untuk :
a. Mengisi Stack1 (PUSH1)
b. Menghapus isi Stack1 (POP1)
c. Mengisi Stack2 (PUSH2)
d. Menghapus isi Stack2 (POP2)

 === Jawaban ===
 // saya menggunakan algoritma pseudocode
a. Mengisi stack1 (PUSH1)
  => - HEAD: 
        Mengisi stack1 (PUSH1)
     - DEKLARASI: 
        DATA : array [1..N] of integer
        TOP1, TOP2, N : integer
        X : integer {input}
     - DESKRIPSI:
        read(X)
        if TOP1 + 1 = TOP2 then
           write("Stack penuh!")
        else
          TOP1 <-- TOP1 + 1
          DATA[TOP1] <-- X
           write("Data berhasil dimasukkan ke Stack1")
           endif
b. Menghapus isi Stack1 (POP1)
 => - HEAD: 
       Menghapus isi Stack1 (POP1)
    - DEKLARASI:
       DATA : array [1..N] of integer
       TOP1 : integer
       X : integer {output}
    - DESKRIPSI:
       if TOP1 = 0 then
         write("Stack1 kosong!")
       else
         X <-- DATA[TOP1]
         TOP1 <-- TOP1 - 1
          write("Data yang dihapus dari Stack1 adalah: ", X)
          endif
          
c. Mengisi Stack2 (PUSH2)
 => - HEAD:
       Mengisi Stack2 (PUSH2)
    - DEKLARASI:
       DATA : array [1..N] of integer
       TOP1, TOP2, N : integer
       X : integer {input}
    - DESKRIPSI:
       read(X)
       if TOP2 - 1 = TOP1 then
           write("Stack penuh!")
       else
         TOP2 <-- TOP2 - 1
         DATA[TOP2] <-- X
          write("Data berhasil dimasukkan ke Stack2")
         endif
      
d. Menghapus isi Stack2 (POP2)
 => - HEAD: 
       Menghapus isi Stack2 (POP2)
    - DEKLARASI:
       DATA : array [1..N] of integer
       TOP2, N : integer
       X : integer {output}
    - DESKRIPSI:
       if TOP2 = N + 1 then
          write("Stack2 kosong!")
       else
         X <-- DATA[TOP2]
         TOP2 <-- TOP2 + 1
         write("Data yang dihapus dari Stack2 adalah: ", X)
         endif
