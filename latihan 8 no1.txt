=== Soal ===
 Tulis algoritma dasar double ended queue untuk kondisi:
a. Inisialisasi
b. Insert sebuah record dari kanan
c. Insert sebuah record dari kiri
d. Delete sebuah record dari kanan
e. Delete sebuah record dari kiri

=== Jawaban ===
a. Inisialisasi
HEAD:
 Inisialisasi
DEKLARASI:
 FONT, REAR
DESKRIPSI:
 FONT <-- -1
 REAR <-- -1

b. Insert sebuah record dari kanan
HEAD:
 Insert sebuah record dari kanan
DEKLARASI:
 RECORD,FONT,REAR
DESKRIPSI:
 if ((FRONT = 0 and REAR = n-1) or (FRONT = REAR + 1)) then
    write("Deque penuh")
 else if (FRONT = -1) then
    FRONT <-- 0
    REAR <-- 0
    DEQUE[REAR] <-- RECORD
 else if (REAR = n-1) then
    REAR <-- 0
    DEQUE[REAR] <-- RECORD
 else
    REAR <-- REAR + 1
    DEQUE[REAR] <-- RECORD
endif

c. Insert sebuah record dari kiri
HEAD:
 Insert sebuah record dari kiri
DEKLARASI:
 RECORD,FRONT,REAR
DESKRIPSI:
 if ((FRONT = 0 and REAR = n-1) or (FRONT = REAR + 1)) then
    write("Deque penuh")
 else if (FRONT = -1) then
    FRONT <-- 0
    REAR <-- 0
    DEQUE[FRONT] <-- RECORD
 else if (FRONT = 0) then
    FRONT <-- n - 1
    DEQUE[FRONT] <-- RECORD
 else
    FRONT <-- FRONT - 1
    DEQUE[FRONT] <-- RECORD
 endif
 
d. Delete sebuah record dari kanan
HEAD: 
 Delete sebuah record dari kanan
DEKLARASI:
 RECORD,FRONT,REAR
DESKRIPSI:

e. Delete sebuah record dari kiri
HEAD:
 Delete sebuah record dari kanan
DEKLARASI:
 RECORD,FRONT,REAR
DESKRIPSI:
 if (FRONT = -1) then
    write("Deque kosong")
 else if (FRONT = REAR) then
    FRONT <-- -1
    REAR <-- -1
 else if (REAR = 0) then
    REAR <-- n - 1
 else
    REAR <-- REAR - 1
 endif
